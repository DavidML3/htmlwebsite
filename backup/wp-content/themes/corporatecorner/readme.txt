A starter theme called CorporateCorner.

== Description ==
CorporateCorner is a clean responsive WordPress Free theme with lots of useful and essential features. It is especially designed for Construction Services, Advising Agencies, Law Firm, Business, Consultancy Agencies, Creative Agencies, General Corporate Websites, Finance, insurance companies and all kind of companies engaged in providing professional or creative services. This theme comes with super powerful and easy page builder. CorporateCorner is very powerful, unique, clean and modern design and this theme is 100% responsive so that you can get powerful experience in any device. It comes with necessary feature for your website such as blog, portfolio, testimonial, etc. This template is built on bootstrap framework, fits any device nicely & perfectly. You can easily customize and build your own website.

== Installation ==
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

= Does this theme support any plugins? =

Corporate Corner includes support for Infinite Scroll in Jetpack.
